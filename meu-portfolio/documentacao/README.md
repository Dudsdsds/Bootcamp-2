# 📄 Documentação

Esta pasta centraliza materiais de apoio, apresentações e relatórios produzidos ao longo dos projetos.

---

## 📂 O que você encontra aqui

| Tipo | Descrição |
|------|-----------|
| Slides | Apresentações em PDF ou PowerPoint |
| Relatórios | Documentos técnicos e acadêmicos |
| Diagramas | Fluxogramas, arquiteturas e modelos |
| Referências | Links e materiais de estudo |

## 📋 Convenção de nomes

Os arquivos seguem o padrão:
```
ANO-MES_nome-descritivo.extensao
Exemplo: 2025-05_apresentacao-tcc.pdf
```
