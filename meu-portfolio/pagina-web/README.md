---

## 🚀 Como publicar no GitHub Pages

1. Acesse seu repositório no GitHub
2. Vá em **Settings → Pages**
3. Em "Source", selecione a branch `main`
4. Em "Folder", selecione `/pagina-web` (ou `/root` se mover o index.html)
5. Clique em **Save** — seu site estará online em minutos!

---

## ✏️ Como personalizar

Abra `index.html` e substitua:
- `Seu Nome` → seu nome real
- `seu-usuario` → seu usuário do GitHub
- `seuemail@email.com` → seu e-mail
- `[Curso]` e `[Instituição]` → seus dados acadêmicos
